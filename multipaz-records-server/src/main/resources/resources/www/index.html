You are playing the role of Admin of State. State is a fictional state, of course, and all individuals here are fictional as well. This page contains the list of individuals registered in State. An individual is uniquely identified by "Utopia id" which is assigned automatically when an individual is first entered in the system. Click on "Utopia id" to review the information on the given individual. Digital identities issued by State are based on the information entered in this system.

You are logged in and thus have power to add a new individual by clicking "New Individual" at the bottom of the page. Make sure to only enter fictional identities here, this system is designed for demo purposes only and as such should not be used to store any real-world personal information!

As you are not logged in, you can view, but not alter this data. Use "Login" button at the top of the page to log in.
